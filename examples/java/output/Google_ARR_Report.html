033 (AA)

033 : H33 Anchor Tag Title For New Windows

Compliance Level: AA

Description

  1. This method conforms to the WCAG rule H33.
  2. This rule identifies the presence of title as for all the links in the page which open in a new window.

Procedure

  • Get all anchor elements that have a title attribute.
  • Check that the title attribute together with the link text describes the purpose of the link.
1 . The anchor with href=https://www.google.com/?gws_rd=ssl# and target=_blank opens in a new window/tab and needs to inform the user. The current label='Send feedback' and description='' is not sufficient. Improve the label (which comes from the aria-labelledby attribute, aria-label attribute or inline text) and/or description (which comes from the aria-describedby attribute or title attribute) to warn the user with the exact verbiage 'opens in new window or tab.' Try <a href='...'>...<span class='clipped'> - opens in new window or tab</span></a>.
